The selectable buttons on this page will not be displayed correctly if the JavaScript setting for the Internet Browser located on the XMB™ is set to "Off".
Please ensure that JavaScript is set to "On" by following the steps below
1. From the XMB™, select Network > Internet Browser.
2. After the internet browser is launched, press △ to display the settings menu.
3. Select Tools and set JavaScript to "On".

The selectable buttons on this page will not display correctly if JavaScript for this Internet browser is disabled in the XMB ™ menu.
Please verify that JavaScript is enabled by following the steps below.
1. From the XMB ™ menu, select Network> Internet Browser.
2. After launching the Internet browser, push △ to display the options menu.
3. Select Tools and enable JavaScript.

The clickable buttons on this page will not display correctly if the JavaScript setting for the XMB ™ Internet Browser is set to 'Off'.
Make sure the JavaScript setting is set to 'Enable' by following the next steps:
1. From the XMB ™ menu, select Network> Broswer for Internet.
2. After launching the Internet Browser, press △ to display the settings menu.
3. Select Tools and then set JavaScript to 'Enable'.

As long as JavaScript is not active for the Internet browser in the XMB ™ menu, the buttons that can be selected on this page will not be displayed correctly.
You can activate JavaScript with the following steps:
1. From the XMB ™ menu, choose Network, then Internet Browser.
2. Once in the web browser, push △ to enter the settings menu.
3. Go to "Tools" and set JavaScript to "On".

The clickable buttons on this page will not display correctly if the Internet browser's JavaScript setting located on the XMB ™ is disabled.
Make sure JavaScript is enabled by following these steps:
1. From the XMB ™, select Network> Internet Browser.
2. When the Internet browser starts, press △ to display the settings menu.
3. Select Tools and in JavaScript select "Yes".

Selectable buttons on this page will not display correctly if JavaScript for the web browser in the XMB ™ menu is disabled.
Make sure JavaScript components are enabled by following these steps:
1. From the XMB ™ menu, select Network> Web Browser.
2. After launching the web browser, push △ to go to the settings menu.
3. Select "Tools" and set "On" for JavaScript.
